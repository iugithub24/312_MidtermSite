Dynamically Changing Web Content

  • - The DOM createElement method
  • - The DOM append method
  • - Template literals
  • - The innerText method

Steps


  1. Create a web document and fill with basic tags.
  2. To the body of the document, add a div element of class demo, where demo is as shown.
  3. Add script tags to the body of the document, just before the body tag.
  4. Now use the DOM method, querySelector to get a reference to the div of class demo.
  1. Use textContent method to add plain text ↓

    [ Let demRef = document.querySelector(“.demo”);

    demoRef.textContent = "Recipe 1, Simple Display"; ]



    You should see this when you render the document →
  1. Add some HTML code. Use innerHTML method ↓

    [ demoRef.innerHTML = `<h3> Recipe1, Simple Display <h3>

    <p> This is an important skill! <p>`]

    Also, modify the style for demo class to have a width/height of 300px.



    When this is rendered you should see →

    <p> This is an important skill! <p>`]
